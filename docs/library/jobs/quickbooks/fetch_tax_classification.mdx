#  Tax Classification - DB Format

The Quickbooks Tax Classification provides information about tax classifications used in financial transactions. It includes details such as tax classification names, domain, and associated information.

### Data headers

| Header | Description |
|---------------------|---------------------------------------------------|
| Name                | Name of the tax classification.                   |
| Description         | Description or additional details about the tax classification.|
| ApplicableTo        | Applicable entities or transactions for the tax classification.|
| Code                | Code associated with the tax classification.    |
| Level               | Level or depth of the tax classification in the hierarchy.|
| ParentRef_value     | Value of the parent reference (if applicable).   |
| ParentRef_name      | Name of the parent reference (if applicable).    |
| Id                  | Unique identifier for the tax classification.    |
