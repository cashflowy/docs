#  Payment Method - Database Format

The Quickbooks Payment Method provides information about various payment methods used in financial transactions. It includes details such as payment method names, types, and associated information.

### Data headers

| Header | Description |
|-------------------------|---------------------------------------------------|
| Name                    | Name of the payment method.                        |
| Active                  | Indicates if the payment method is active (true/false).|
| Type                    | Type or category of the payment method.            |
| Domain                  | Domain information related to the payment method. |
| Sparse                  | Indicates if the payment method is sparse (true/false).|
| Id                      | Unique identifier for the payment method.          |
| SyncToken               | Token for synchronization purposes.                |
| MetaData_CreateTime     | Timestamp indicating the creation time.           |
| MetaData_LastUpdatedTime| Timestamp indicating the last update time.        |
