### Data headers

| Header | Description |
|-----------------------------|---------------------------------------------------|
| Name                        | Name of the tax rate.                             |
| Description                 | Description or additional details about the tax rate.|
| Active                      | Indicates if the tax rate is active (true/false).|
| RateValue                   | Percentage value of the tax rate.                 |
| SpecialTaxType              | Special type of tax, if applicable.               |
| DisplayType                 | Type of display for the tax rate.                 |
| Domain                      | Domain information related to the tax rate.       |
| Sparse                      | Indicates if the tax rate is sparse (true/false).|
| SyncToken                   | Token for synchronization purposes.                |
| AgencyRef_value             | Value of the agency reference associated with the tax rate.|
| MetaData_CreateTime         | Timestamp indicating the creation time.           |
| MetaData_LastUpdatedTime    | Timestamp indicating the last update time.        |
| Id                          | Unique identifier for the tax rate.              |
