#  Accounts - Database Format


The QuickBooks List Accounts Details dataset provides essential information about accounts, including unique identifiers, names, types, currency codes, balances, default status, and creation/modification timestamps.

### Data headers

| Headers                      | Description                                                        |
|------------------------------|--------------------------------------------------------------------|
| Account       | General category of the account (example: Asset, Liability). |
| Type          | Type of the account (example: Bank, Expense).   |
| Detail Type   | Specific subtype or detail type of the account. |
| Description   | Brief description of the account.               |
| Balance   | Brief balance of the account.               |
