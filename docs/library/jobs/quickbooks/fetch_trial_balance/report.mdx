#  Trial Balance - Report


The QuickBooks Trial Balance is a financial report that summarizes the balances of all general ledger accounts at a specific point in time. It ensures the equality of debits and credits, providing a fundamental overview of a company's financial standing.

### Data headers

| Header | Description |
|----|----|
| Accounts | Name of the related account |
| Accounts Id | Quickbooks Id of the related account |
| Total for month |  Total account balance for a specific month |
| Total |  Total account balance  |

### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select transactions with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select transactions with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
