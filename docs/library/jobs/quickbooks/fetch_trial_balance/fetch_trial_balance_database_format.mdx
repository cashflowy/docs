### Data headers

| Headers                      | Description                        |
|----------------|--------------------------------------------------|
| Account Name   | Name of the account.                             |
| Account ID     | Unique identifier for the account.               |
| Month          | Specific month for the trial balance details.    |
| Transaction Type| Type of the transaction (e.g., Debit, Credit).  |
| Value          | Transaction value associated with the account.   |
