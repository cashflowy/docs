### Data headers

| Header | Description |
|----|----|
| SourceCurrencyCode     | Code of the source currency.                     |
| TargetCurrencyCode     | Code of the target currency.                     |
| Rate                   | Exchange rate between the source and target currencies.|
| AsOfDate               | Date when the exchange rate is applicable.      |
| MetaData_LastUpdatedTime| Timestamp indicating the last update time of metadata.|
