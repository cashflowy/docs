### Data headers

| Header             | Description                             |
|----------|---------------------------------------------------|
| Adjustment                                | Adjustment value associated with the journal entry.                                           |
| domain                                    | The domain to which the journal entry belongs.                                                |
| sparse                                    | A boolean indicating whether the entry is sparse.                                             |
| SyncToken                                 | Token for synchronizing updates.                                                              |
| DocNumber                                 | Document number associated with the journal entry.                                           |
| TxnDate                                   | Transaction date of the journal entry.                                                       |
| Line_Amount                               | Amount associated with the line item in the journal entry.                                   |
| Line_DetailType                           | Detail type of the line item in the journal entry.                                           |
| Line_JournalEntryLineDetail_AccountRef_value | Value of the account reference for the line item in the journal entry.                    |
| Line_JournalEntryLineDetail_AccountRef_name | Name of the account reference for the line item in the journal entry.                     |
| Line_JournalEntryLineDetail_PostingType | Posting type associated with the line item in the journal entry.                              |
| Line_JournalEntryLineDetail_Entity_Type | Type of entity associated with the line item in the journal entry.                           |
| Line_JournalEntryLineDetail_Entity_EntityRef_value | Value of the entity reference for the line item in the journal entry.                    |
| Line_JournalEntryLineDetail_Entity_EntityRef_name | Name of the entity reference for the line item in the journal entry.                      |
| CurrencyRef_value                         | Value of the currency reference for the journal entry.                                       |
| CurrencyRef_name                          | Name of the currency reference for the journal entry.                                        |
| MetaData_CreateTime                       | Timestamp indicating the creation time of the journal entry.                                 |
| MetaData_LastUpdatedTime                  | Timestamp indicating the last update time of the journal entry.                               |
| Id                                        | QuickBooks Id for the journal entry.                                                      |
| Line_Id                                   | QuickBooks Id for the line item in the journal entry.                                    |
