### Data headers

| Header             | Description                             |
|---------------------|------------------------------------------------------------|
| Date                | Date of the transaction.                                    |
| Transaction Type    | Type of the transaction (e.g., sale, purchase, etc.).      |
| Num                 | Transaction number or identifier.                          |
| Name                | Name associated with the transaction.                      |
| Memo/Description    | Memo or description providing additional information.     |
| Account             | Account associated with the transaction.                   |
| Debit               | Debit amount for the transaction (if applicable).         |
| Credit              | Credit amount for the transaction (if applicable).        |
