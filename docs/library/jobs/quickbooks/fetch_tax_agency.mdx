### Data headers

| Header | Description |
|-----------------------------|---------------------------------------------------|
| DisplayName                 | Display name of the tax agency.                    |
| TaxTrackedOnPurchases       | Indicates if tax is tracked on purchases (true/false).|
| TaxTrackedOnSales           | Indicates if tax is tracked on sales (true/false).    |
| Domain                      | Domain information related to the tax agency.     |
| Sparse                      | Indicates if the tax agency is sparse (true/false).|
| SyncToken                   | Token for synchronization purposes.                |
| MetaData_CreateTime         | Timestamp indicating the creation time.           |
| MetaData_LastUpdatedTime    | Timestamp indicating the last update time.        |
| Id                          | Unique identifier for the tax agency.              |
