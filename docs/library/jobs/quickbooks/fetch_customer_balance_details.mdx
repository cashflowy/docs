Quickbooks Customer Balance Details provides information about the balances associated with customers in financial transactions. It includes details such as customer names, invoiced amounts, amounts received, closing balances etc.

### Data headers

| Header             | Description                                             |
|------------------|---------------------------------------------------|
| Date             | Date of the financial transaction.                 |
| Transaction Type | Type or category of the financial transaction.     |
| Num              | Transaction number or identifier.                 |
| Due Date         | Date by which the payment is due.                 |
| Amount           | Total amount of the financial transaction.        |
| Open Balance     | Remaining balance to be paid by the customer.     |
| Balance          | Overall balance of the customer account.           |
