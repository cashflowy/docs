### Data headers 

| Header                   | Description                                                         |
|--------------------------|---------------------------------------------------------------------|
| id                       | Unique identifier for the object.                                   |
| object                   | Type of object (e.g., data).                                        |
| data_available_end       | End date for data availability.                                     |
| data_available_start     | Start date for data availability.                                   |
| default_columns/0 to 23  | Default columns for data (assuming an array-like structure).        |
| livemode                 | Indicates whether the object is in live mode.                       |
| name                     | Name or title associated with the object.                           |
| updated                  | Date and time of the last update.                                   |
| version                  | Version information for the object.                                 |
