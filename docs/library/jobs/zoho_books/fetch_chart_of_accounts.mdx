### Data headers

| Header | Description |
|----|----|
| account_name | Name of the account |
| account_type | Type of the account (e.g., other_current_asset) |
| description | Description of the account |
| is_active | Indicates if the account is active (true/false) |
| account_code | Account code (if available) |
| account_id | Zoho Books ID of the account |
| parent_account_name | Name of the parent account (if applicable) |
| parent_account_id | Zoho Books ID of the parent account (if applicable) |
| created_time | Time the account was created in Zoho Books |
| last_modified_time | Time the account was last modified in Zoho Books |
| is_user_created | Indicates if the account was user-created (true/false) |
| is_system_account | Indicates if the account is a system account (true/false) |
| is_standalone_account | Indicates if the account is standalone (true/false) |
| can_show_in_ze | Indicates if the account can show in Zoho Expense (true/false) |
| is_child_present | Indicates if the account has child accounts (true/false) |
| has_attachment | Indicates if the account has attachments (true/false) |
| depth | Depth of the account in the account hierarchy (e.g., 0) |
| child_count | Number of child accounts (if applicable) |
