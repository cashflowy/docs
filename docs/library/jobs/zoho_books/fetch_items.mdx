### Data headers

| Header                      | Description                                               |
|-----------------------------|-----------------------------------------------------------|
| name                        | Unique identifier for the item.                            |
| item_name                   | The name or title of the item.                             |
| unit                        | Unit of measurement for the item.                          |
| status                      | Current status of the item.                                |
| source                      | Source information for the item.                           |
| is_linked_with_zohocrm      | Indicates if the item is linked with Zoho CRM.             |
| description                 | Detailed description of the item.                          |
| rate                        | Price or rate associated with the item.                    |
| tax_name                    | The name of the tax applied to the item.                   |
| tax_percentage              | The percentage of tax applied to the item.                 |
| purchase_account_name       | The name of the purchase account associated with the item. |
| account_name                | The name of the account associated with the item.          |
| purchase_description        | Description for purchase transactions related to the item. |
| purchase_rate               | Purchase rate for the item.                                |
| item_type                   | The type or category of the item.                          |
| product_type                | Type of product (if applicable).                           |
| is_taxable                  | Indicates if the item is taxable.                          |
| tax_exemption_code          | Code for tax exemption (if applicable).                   |
| has_attachment              | Indicates if the item has attachments.                     |
| sku                         | Stock Keeping Unit (SKU) for the item.                     |
| image_name                  | Name of the item's associated image.                       |
| image_type                  | Type of the item's associated image.                       |
| created_time                | Timestamp indicating when the item was created.            |
| last_modified_time          | Timestamp indicating when the item was last modified.     |
| hsn_or_sac                  | Harmonized System Nomenclature (HSN) or Service Accounting Code (SAC) for the item. |
| show_in_storefront          | Indicates if the item is visible in the storefront.        |
| cf_payment_type             | Custom field for payment type.                             |
| cf_payment_type_unformatted | Unformatted custom field for payment type.                 |
| item_id                     | Zoho books id for the item.                            |
| zcrm_product_id             | Zoho books id for ZCRM product.                             |
| tax_id                      | Zoho books id for the associated tax.                         |
| account_id                  | Zoho books id for the associated account.                     |
| purchase_account_id         | Zoho books id for the associated purchase account.           |
| tax_exemption_id            | Zoho books id for the associated tax exemption.               |
| image_document_id           | Zoho books id for the associated image document.             |



### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
