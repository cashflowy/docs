### Data headers

| Header                       | Description                                              |
|------------------------------|----------------------------------------------------------|
| rate_formatted               | Formatted representation of the exchange rate.            |
| rate                         | Numerical value representing the exchange rate.           |
| effective_date_formatted     | Formatted effective date of the exchange rate.            |
| effective_date               | Numerical representation of the effective date.           |
| is_market_closed_rates       | Indicates whether market rates are closed.               |
| currency_code                | The currency code associated with the exchange rate.      |
| currency_id                  | Zoho books id for the currency in Zoho Books.        |
