### Data headers

| Header               | Description                                           |
|----------------------|-------------------------------------------------------|
| tax_name             | Name of the tax                                       |
| tax_percentage       | Percentage rate of the tax                            |
| tax_type             | Type of tax                           |
| tax_specific_type    | Specific type of tax (if applicable)                  |
| is_inactive          | Indicates whether the tax is inactive                |
| is_default_tax       | Indicates whether the tax is set as the default       |
| is_editable          | Indicates whether the tax is editable                |
| tax_specification    | Additional specification or details about the tax     |
| diff_rate_reason     | Reason for any difference in the tax rate             |
| start_date           | Start date of the tax                                 |
| end_date             | End date of the tax                                   |
| last_modified_time   | Time when the tax was last modified                   |
| status               | Current status of the tax                             |
| tax_id               | Zoho books id for the tax                         |



### Filteration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
