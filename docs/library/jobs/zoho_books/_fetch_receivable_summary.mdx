### Data headers

| Header             | Description                                           |
|--------------------|-------------------------------------------------------|
| customer_name      | Name of the customer associated with the receivable   |
| transaction_date   | Date of the receivable transaction                    |
| transaction_number | Unique identifier for the receivable transaction      |
| reference_number   | Reference number associated with the receivable transaction |
| status             | Status of the receivable transaction (e.g., pending, completed) |
| transaction_type   | Type of the receivable transaction (e.g., invoice, payment) |
| bcy_total          | Total amount in the base currency                      |
| bcy_balance        | Remaining balance in the base currency                 |
| fcy_total          | Total amount in the foreign currency                   |
| fcy_balance        | Remaining balance in the foreign currency              |
| salesperson_name   | Name of the salesperson involved in the transaction    |
| currency_code      | Currency code for the transaction                      |
| transaction_id     | Unique identifier for the receivable transaction        |
| customer_id        | Unique identifier for the customer associated with the receivable |
| currency_id        | Unique identifier for the currency associated with the receivable |





### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
