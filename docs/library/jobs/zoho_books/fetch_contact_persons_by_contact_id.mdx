### Data headers

| Header                           | Description                                          |
|----------------------------------|------------------------------------------------------|
| salutation                       | Salutation or title for the contact person.         |
| first_name                       | First name of the contact person.                    |
| last_name                        | Last name of the contact person.                     |
| email                            | Email address of the contact person.                 |
| phone                            | Phone number of the contact person.                  |
| mobile                           | Mobile number of the contact person.                 |
| department                       | Department or division of the contact person.        |
| designation                      | Job title or designation of the contact person.      |
| skype                            | Skype ID of the contact person.                      |
| fax                              | Fax number of the contact person.                    |
| is_primary_contact               | Indicates if the contact person is the primary contact. |
| is_added_in_portal               | Indicates if the contact person is added in the portal. |
| can_invite                       | Indicates if the contact person can be invited.     |
| is_portal_invitation_accepted    | Indicates if the portal invitation is accepted by the contact person. |
| contact_person_id                | Zoho books id for the contact person.            |
