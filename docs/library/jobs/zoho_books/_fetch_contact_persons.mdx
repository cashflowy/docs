Contact Persons in Zoho Books are individuals associated with a contact. This includes details such as their name, contact information, role, and status. The information helps manage communication and responsibilities within the contacts associated with Zoho Books.

### Data headers

| Header              | Description                                           |
|---------------------|-------------------------------------------------------|
| contact_name        | Name of the associated contact                        |
| salutation          | Salutation or title for the contact person            |
| first_name          | First name of the contact person                      |
| last_name           | Last name of the contact person                       |
| email               | Email address of the contact person                   |
| phone               | Phone number of the contact person                    |
| mobile              | Mobile number of the contact person                   |
| department          | Department or division the contact person belongs to  |
| designation         | Job title or role of the contact person               |
| skype               | Skype ID of the contact person                        |
| fax                 | Fax number of the contact person                      |
| currency_code       | Currency code associated with the contact person      |
| created_time        | Time when the contact person was created              |
| is_primary_contact  | Indicates whether the contact person is the primary contact |
| contact_id          | Unique identifier for the associated contact           |
| contact_person_id   | Unique identifier for the contact person              |
