Time to Get Paid in Zoho Books refers to the duration it takes for invoices to be paid by customers. This metric tracks the average time between issuing an invoice and receiving payment

### Data headers

| Header          | Description                                   |
|------------------|-----------------------------------------------|
| customer_name    | Name of the customer                           |
| range_0-15       | Total outstanding amount aged 0-15 days        |
| range_16-30      | Total outstanding amount aged 16-30 days       |
| range_31-45      | Total outstanding amount aged 31-45 days       |
| range_above45    | Total outstanding amount aged above 45 days    |
| currency_code    | Currency code associated with the amounts     |
| customer_id      | Unique identifier for the customer            |

### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
