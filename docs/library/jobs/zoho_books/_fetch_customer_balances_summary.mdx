### Data headers

| Header            | Description                                              |
|-------------------|----------------------------------------------------------|
| customer_name     | The name of the customer associated with the summary.     |
| invoiced_amount   | The total amount invoiced to the customer.                |
| amount_received   | The total amount received from the customer.              |
| closing_balance   | The remaining balance after considering invoiced amounts and received payments. |
| customer_id       | Zoho books id for the customer in the business's system. |




### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|                                                                                  
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
