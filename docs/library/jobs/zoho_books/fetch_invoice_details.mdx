### Data headers

| Header            | Description                                               |
|-------------------|-----------------------------------------------------------|
| date              | The date of the invoice.                                   |
| status            | The status of the invoice.                                 |
| due_date          | The due date for payment.                                  |
| invoice_number    | The unique identifier for the invoice.                     |
| reference_number  | A reference number associated with the invoice.            |
| customer_name     | The name of the customer.                                  |
| bcy_total         | The total amount in base currency (BCY).                  |
| bcy_balance       | The remaining balance in base currency.                   |
| project_names     | Names of the projects associated with the invoice.        |
| currency_code     | The currency code used for the transaction.               |
| price_precision   | Precision of the price.                                   |
| is_emailed        | A boolean indicating whether the invoice has been emailed.|
| reminders_sent    | The number of reminders sent for payment.                 |
| exchange_rate     | The exchange rate used for currency conversion.           |
| type              | The type of the invoice.                                   |
| invoice_id        | Zoho books id for the invoice.                    |
| salesperson_id    | The identifier for the salesperson associated with the invoice. |
| currency_id       | The identifier for the currency used.                     |
| customer_id       | Zoho books id for the customer.                   |
| ym                | Calculated year-month based on the invoice date.          |



### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select invoice with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select invoices with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
