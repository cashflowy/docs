### Data headers

| Header | Description |
|----|----|
| customer_name | Name of the customer |
| status | Status (e.g., active) |
| place_of_contact_formatted | Formatted place of contact |
| gst_treatment | GST treatment |
| gst_no | GST number |
| currency | Currency |
| outstanding_receivable_amount | Outstanding receivable amount |
| unused_credits_receivable_amount | Unused credits receivable amount |
| outstanding_receivable_amount_bcy | Outstanding receivable amount in base currency |
| unused_credits_receivable_amount_bcy | Unused credits receivable amount in base currency |
| payment_terms_label | Formatted payment terms label |
| customer_sub_type | Customer subtype (e.g., business) |
| tax_treatment |	Tax treatment |
| created_time | Time the contact was created in Zoho Books |
| last_modified_time | Time the contact was last modified in Zoho Books |
| contact_id | zoho books id of the customer related to this payment |
