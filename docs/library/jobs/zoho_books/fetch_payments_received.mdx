Zoho Books Payment Received records and organizes received payments. It includes crucial information like payment date, method, customer details, and associated invoices.

### Data headers

| Header                     | Description                                           |
|----------------------------|-------------------------------------------------------|
| date                       | Date of the payment received                           |
| payment_number             | Payment number                                        |
| payment_mode               | Payment mode used for the transaction                  |
| customer_name              | Name of the customer                                  |
| invoice_numbers            | List of invoice numbers related to the payment         |
| invoice_number             | Single invoice number related to the payment           |
| applied_invoice_amount     | Amount applied to the invoice in transaction currency |
| applied_invoice_bcy_amount | Amount applied to the invoice in base currency        |
| applied_date               | Date when the payment was applied                      |
| amount                     | Amount of the payment received in transaction currency|
| bcy_amount                 | Amount of the payment received in base currency        |
| unused_amount              | Unused amount after applying to invoices               |
| bcy_unused_amount          | Unused amount in base currency after applying to invoices |
| account_name               | Name of the associated account                         |
| description                | Description of the payment                             |
| product_description        | Description of the associated product                  |
| reference_number           | Reference number for the payment                       |
| last_four_digits           | Last four digits of the payment method                 |
| type                       | Type of the payment (e.g., advance payment)            |
| bcy_refunded_amount        | Refunded amount in base currency                       |
| is_advance_payment         | Indicates if the payment is an advance payment         |
| has_attachment             | Indicates if the payment has attachments              |
| documents                  | List of documents attached to the payment              |
| tax_account_name           | Name of the associated tax account                     |
| tax_amount_withheld        | Amount withheld for tax                               |
| currency_code              | Currency code used in the transaction                  |
| payment_method             | Method used for the payment                            |
| place_of_supply            | Place of supply for tax purposes                       |
| payment_id                 | Zoho Books ID for the payment                          |
| retainerinvoice_id         | Zoho Books ID for the related retainer invoice         |
| account_id                 | Zoho Books ID for the associated account               |
| customer_id                | Zoho Books ID for the related customer                 |
| gateway_transaction_id     | Transaction ID from the payment gateway                |
| tax_account_id             | Zoho Books ID for the associated tax account           |
| ym                         | Year and month of the entry in YYYY-MM format          |
