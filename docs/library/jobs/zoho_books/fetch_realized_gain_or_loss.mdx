### Data headers

### Zoho Books: Line Items Table

| Header                      | Description                                            |
|-----------------------------|--------------------------------------------------------|
| transaction_date            | The date of the transaction.                            |
| realized_amount             | The actual amount realized from the transaction.        |
| exchange_rate               | The rate used for currency exchange.                    |
| transaction_type            | The type of transaction.                                |
| currency_code               | The currency code used for the transaction.            |
| gain_or_loss                | The calculated gain or loss from the transaction.       |
| transaction_type_formatted | The formatted representation of the transaction type.   |
| transaction_id              | Zoho books id for the transaction.                  |
| customer_id                 | Zoho books id for the associated customer.         |

        

### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
