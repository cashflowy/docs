This job fetches expense details from zoho books. According to Zoho books, expense is an alternative to bills where the expense and the payment is recorded at the same time.

### Data headers

| Headers              | Description                                           |
|---------------------|-------------------------------------------------------|
| date                | Date of the transaction.                              |
| status              | Status of the transaction.                            |
| transaction_number  | Number for the transaction.                |
| distance            | Distance associated with the transaction.             |
| vendor_name         | Name of the vendor involved in the transaction.       |
| account_name        | Name of the account related to the transaction.       |
| customer_name       | Name of the customer involved in the transaction.     |
| amount              | Transaction amount without tax.                       |
| amount_with_tax     | Transaction amount including tax.                     |
| transaction_type    | Type of financial transaction.                        |
| entity_type         | Type of entity associated with the transaction.      |
| transaction_id      | Zoho books id for the transaction.                |
| vendor_id           | Zoho books id for the vendor.                     |
| customer_id         | Zoho books id for the customer.                   |
| ym                  | Year and month of the transaction (formatted).        |


### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select transactions with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select transactions with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
