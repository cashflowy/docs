### Data headers

| Header            | Description                                           |
|-------------------|-------------------------------------------------------|
| date              | Date of the refund transaction                        |
| refund_mode       | Mode of the refund (e.g., credit note, bank transfer)  |
| entity_type       | Type of entity associated with the refund              |
| entity_number     | Entity number associated with the refund               |
| vendor_name       | Name of the vendor receiving the refund                |
| bcy_amount        | Refund amount in base currency                         |
| reference_number  | Reference number for the refund                        |
| fcy_amount        | Refund amount in foreign currency                      |
| description       | Description of the refund transaction                 |
| currency_code     | Currency code for the refund transaction              |
| entity_id         | Unique identifier for the associated entity            |
| refund_id         | Unique identifier for the refund transaction           |
| vendor_id         | Unique identifier for the vendor receiving the refund  |



### Filteration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
