Account Type Summary retrieves a concise overview of various account types, detailing their credits, debits, and total amounts. Use this feature to quickly assess the financial distribution across different account categories in your records.

### Data headers

| Header          | Description                                           |
|-----------------|-------------------------------------------------------|
| account_type    | Type of the account (e.g., assets, liabilities, income, expenses) |
| credit          | Total credit amount associated with the account type    |
| debit           | Total debit amount associated with the account type     |
| account_type_id | Unique identifier/code for the account type             |
| ym              | Year and month corresponding to the entry              |




### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
