Credit cards, cash accounts, bank accounts etc can be found under this module. This fetches all the accounts created in zoho. 

Some of the thing that you can do with this data
- visualise balances in each of these accounts on a daily basis
- make decisions on organising your accounts better
- reconcile with actual bank statements

### Data headers

| Header | Description |
|----|----|
| account_name | Name of the account |
| account_type | Type of the account. Can be - `bank`, `credit_card`, `cash`, `payment_clearing` |
| account_number | Account number |
| currency_code | Currency of this account |
| book_balance | Balance in this account as per book |
| bcy_book_balance | Balance in this account as per books in company's default currency |
| is_active | Is this account active |
| live_balance | Balance as per auto refreshed data |
| uncategorized_transactions | No of transactions that are uncategorized |
| feeds_last_refresh_date | Date this was auto refreshed by zoho books |
| refresh_status | Status of auto refresh by zoho books |
| account_id | zoho books id of the account |

### Filteration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| is_active | Select bank Aaccounts by status. Options: `True`,`False` |
