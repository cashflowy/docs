This job represents user-generated textual entries that provide additional information, explanations, or updates related to various activities. They contain details such as content, POR number, user name, email, creation and update timestamps, along with unique identifiers.

### Data headers

| Header     | Description |
|--------|-------------|
| comment    | Comment content associated with a Purchase Order Request (POR) |
| por_no     | Purchase Order Request (POR) number |
| user       | User's name who posted the comment |
| user_email | User's email address who posted the comment |
| createdAt  | Timestamp indicating when the comment was created |
| updatedAt  | Timestamp indicating the last update time of the comment |
| id         | Unique identifier for the comment |
| por_id     | Unique identifier for the associated Purchase Order Request (POR) |




### Filtration options

This job can restrict the data that is visible in spreadsheets. Use the following parameters to restrict the data.

| Parameter | Options |
|----|----|
| start_date | Select bills with billing date on or after `start_date`. Format - `YYYY-MM-DD` |
| end_date | Select bills with billing date on or before `end_date`. Format - `YYYY-MM-DD` |
