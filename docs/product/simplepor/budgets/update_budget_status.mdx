Budget status can be: 
- active 
- paused
- archived

| Status   | Details (what this status means)                                                   |
|----------|------------------------------------------------------------------------------------|
| active   | This budget is active. Users can create a PORs against this budget.                |
| paused   | This budget is temporarily stopped. Users cannot create PORs against this budget.  |
| archived | This budget is permanently stopped. Users cannot create PORs against this budget.  |
