| Field         | Details                                                                                                                                                                                   |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| code          | is used to reconcile this budget with ERP or accounting tool. Also used by users to refer to a budget instead of the name.                                                                |
| name          | This is a human readable name for the budget.                                                                                                                                             |
| Budget amount | Amount set for this budget. Available amount remaining in this budget will be Budget amount(this value) - Committed - Paid out                                                            |
| Committed     | From the budget amount this amount is committed to be paid but not yet paid. When a POR is approved this value is updated(increases). When a POR is marked as paid, this value decreases  |
| Paid out      | From the budget amount this amount is already paid out. When a POR is marked as paid, this value is updated(increases).                                                                   |
| Approver 1    | First approver for this budget. Typically a manager                                                                                                                                       |
| Approver 2    | Second approver for this budget. Typically a senior manager                                                                                                                               |
